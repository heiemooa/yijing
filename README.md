Project documentation with Markdown.

## 安装

```
pip install mkdocs
```

## 新建项目

```
mkdocs new my-project
```

## 启动:

```bash
mkdocs serve
```

本地运行： [http://localhost:8000](http://localhost:8000)

- 部署 Github Page
```
mkdocs gh-deploy
```

- 部署服务器

```
mkdocs build
```
这将生成一个包含html文件的site目录
