Project documentation with Markdown.

## 安装

```
pip install mkdocs
```

## 新建项目

```
mkdocs new my-project
```

## 启动:

```bash
mkdocs serve
```

本地运行： [http://localhost:8000](http://localhost:8000)

- 打包部署

```
mkdocs build
```
